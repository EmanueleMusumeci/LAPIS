The LAPIS Pipeline

Bridging the gap between LLM flexibility and symbolic planning rigor.

🔧

Domain Synthesis

Translates NL task descriptions into structured PDDL domain files with VAL-guided syntax correction.

→
🔍

Adequacy Check

Verifies domain expressivity via Chain-of-Thought reasoning to identify missing predicates or actions.

→
📋

Problem Gen

Generates specific problem instances using dynamic schema injection from the synthesised domain.

→
⚙️

Refinement Loop

Iteratively refines PDDL via symbolic feedback until a valid plan is found by a classical solver.

Provable Correctness

Uses VAL (PDDL Validator) to ensure that every generated plan is logically sound and executable.

Zero-Shot Deployment

Handles entirely new domains without requiring manual PDDL modeling or expert-in-the-loop intervention.

Complex Reasoning

Combines the broad semantic knowledge of LLMs (GPT-4o/Claude) with the precision of classical search.

Scalability

Successfully plans tasks with long horizons (>80 steps) where pure autoregressive models fail.

Benchmarked Performance

LAPIS consistently outperforms single-shot translation baselines on IPC domains.
